# ONSOFA.ai
ONSOFA.ai under construction!

This repository contains a very small, static website that demonstrates how to host a multi‑page site on GitHub Pages using plain HTML/CSS.

- **Home** – Overview of the lab
- **Projects** – Current and past research projects
- **Publications** – Key papers
- **Products** – Commercial offerings
- **Contact Us** – How to reach us

## How to Deploy

1. Fork or clone this repo.
2. Push the branch to GitHub (default `main`).
3. In Settings → Pages → Source, select `main` → `/ (root)`.
4. Afterthat, this site will be live at `https://danielsaatchi.github.io/onsofa/`.

Happy hacking! 🚀
